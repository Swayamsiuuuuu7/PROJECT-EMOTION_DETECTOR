# PROJECT-EMOTION_DETECTOR
# By Swayam

This is a real-time emotion detection web app built using Flask, OpenCV, and TensorFlow.

## Features

- Detects facial emotions live from your webcam
- Classifies into: Angry, Disgust, Fear, Happy, Sad, Surprise, Neutral
- Clean web interface (HTML + Flask)
- Works on desktop and mobile (when hosted online)

## Tech Stack

- Python
- Flask
- OpenCV
- TensorFlow
- HTML/CSS (for the frontend)

## How to Run Locally

1. Clone this repo
2. Install dependencies from `requirements.txt`
3. Run:

```bash
python app.py
# Open http://127.0.0.1:5000 in your browser
